# KCP-Socket
This is a  KCP+UDPSocket Wrapper.
You can use KCP like UdpSocket.

Because it is simple, you can use it for Unity+Lan Communications.

It refers to kcp-cssharp:https://github.com/limpo1989/kcp-csharp
